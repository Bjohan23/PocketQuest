Idioma: Español
Rol: Actúa como un ingeniero senior en desarrollo móvil con React Native y TypeScript.
Objetivo: Crear una aplicación móvil educativa y experimental que explore arquitectura modular, separación de contextos y diseño UI multi-modo.

CONTEXTO

La aplicación se llama Pocket Quest y es un proyecto educativo para aprender arquitectura móvil, navegación avanzada y preparación para mensajería privada.

La app contiene dos modos funcionales independientes:

Un modo de juego casual

Un modo de comunicación privada

Ambos modos conviven dentro de la misma aplicación, pero están claramente separados a nivel de estado y navegación.

El objetivo es arquitectura, UI y estructura, no evasión ni ocultamiento.

TECNOLOGÍAS
